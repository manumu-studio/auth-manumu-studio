## Architecture Decisions

| Decision | Why |
|----------|-----|
| Enterprise password rules (8–128 chars, upper, lower, number, special, no spaces) | Aligns with Microsoft/Google standards for OAuth/OIDC provider |
| Tooltip instead of always-visible checklist | Keeps signup form compact; requirements on demand |
| Dropdown inside inputWrapper | Ensures correct positioning relative to Country input |
| autocomplete="nope" + form autocomplete="off" | Reduces browser address autofill on Country field |
| 0.12s theme transition | Fast and subtle; avoids jarring flash when system preference changes |
| devIndicators: false | Workaround for Next.js 15 SegmentViewNode / React Client Manifest bug |
