# Continuation Prompts

Use this folder for self-contained prompts that resume paused workstreams.

## Convention

- Active prompts: `CP-NNN-short-slug.md`
- Completed/old prompts: `archive/`
- Registry: `CONTINUATION_REGISTRY.md` at repo root when a session is closed.

Continuation prompts should include current state, relevant files, blockers,
and exactly one next action.
